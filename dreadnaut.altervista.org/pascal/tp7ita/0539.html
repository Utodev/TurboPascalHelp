 Concat (funzione)
 ------------------
Concatena una sequenza di stringhe.

 Dichiarazione:
 function Concat(s1 [, s2,..., sn]: String): String;

 Target:
Windows, Real, Protected

 Note:
Ogni parametro e' un'espressione di tipo stringa. Il risultato e' la
concatenazione di tutte le stringhe passate come parametro. Se la stringa
risultante e' piu' lunga di 255 caratteri, questa e' troncata dopo il
255esimo carattere. Usare l'operatore piu' (+) produce lo stesso risultato
della funzione Concat:

 S := 'ABC' + 'DEF';

 Vedere anche:
 Copy

 Pos

 Codice esempio:



  {Concat.PAS}

 {Esempio per la funzione Concat.}

 var
   S: String;
 begin
   S := Concat('ABC', 'DEF'); { 'ABCDE' }
 end.
