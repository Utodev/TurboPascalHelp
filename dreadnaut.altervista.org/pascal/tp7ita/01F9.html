 +--------------------+
 | File access denied |
 +--------------------+

     Restituito da  | Se e' stato riscontrato questo problema
 =================================================================
    Reset o Append  | Se FileMode e' settato per permettere la scrittura ma il nome assegnato
                    | alla variabile indica un file o una directory a sola lettura.
           Rewrite  | Se la directory e' piena o se il nome assegnato alla variable specifica
                    | un file o una directory esistente o a sola lettura.
            Rename  | Se il nome assegnato alla variabile file indica una directory, o se
                    | il nuovo nome corrisponde ad un file gia' esistente.
             Erase  | Se il nome assegnato alla variabile file indica una directory
                    | o un file a sola lettura.
             MkDir  | Se un file con lo stesso nome esiste gia' nella directory, o se non c'e'
                    | piu' spazio nella directory, o se il percorso indica un device.
             RmDir  | Se la directory non è vuota, o se il path non indica una directory,
                    | o se il path specifica la directory radice di una unita'.
  Read o BlockRead  | (Su un file tipato o non tipato) il file non e' aperto per la lettura.
 Write o BlockWrite | (Su un file tipato o non tipato) il file non e' aperto per la scrittura.
