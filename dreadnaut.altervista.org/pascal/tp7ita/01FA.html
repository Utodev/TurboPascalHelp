 +---------------------+
 | Invalid file handle |
 +---------------------+

Questo errore viene restituito se si passa ad una funzione
DOS un handle per file non valido.

Questo errore non dovrebbe mai avvenire; se accade, significa
che la variabile file è stata in qualche modo sovrascritta.
