 Pos (function)
 ---------------
Cerca una sottostringa all'interno di una stringa.

 Dichiarazione:
 function Pos(Substr: String; S: String): Byte;

 Target:
Windows, Real, Protected

 Note:
Substr ed S sono espressione di tipo stringa. Pos cerca Substr all'interno
di S e ritorna il valore intero che indica la posizione del primo carattere
di Substr dentro S. Se Substr non viene trovata, Pos restituisce zero.

 Vedere anche:
 Delete
 Concat
 Copy
 Insert
 Length

Codice esempio:



 {Pos.PAS}

 {Esempio per la funzione Pos.}

 var S: String;
 begin
  S := '   123.5';
  { Converte gli spazi in zeri }
  while Pos(' ', S) > 0 do
    S[Pos(' ', S)] := '0';
 end.
