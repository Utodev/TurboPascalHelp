 Insert (procedura)
 -------------------
Inserisce una sottostringa in una stringa.

 Dichiarazione:
 procedure Insert(Source: String; var S: String; Index: Integer);

 Target:
Windows, Real, Protected

 Note:
Source e' un'espressione di tipo stringa. S e' una variabile di tipo stringa
di qualsiasi lunghezza. Index e' un'espressione di tipo intero. Insert
inserisce Source dentro S all'Index-esima posizione. Se le stringa risultante
e' piu' lunga di 255 caratteri questa viene troncata dopo il 255esimo
carattere.

 Vedere anche:
 Concat
 Copy
 Delete
 Length
 Pos

 Sample Code:



  {Insert.PAS}

 {Esempio per la procedura Insert.}

 var
   S: String;
 begin
   S := 'Honest Lincoln';
   Insert('Abe ', S, 8);  { 'Honest Abe Lincoln' }
 end.
