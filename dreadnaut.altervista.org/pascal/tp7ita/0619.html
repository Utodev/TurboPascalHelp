 set (reserved word)
 ---------------------

 Sintassi:
   set of type

 Note:
Il tipo base di un insieme (set) deve essere un tipo ordinal con non
piu' di 256 valori.

I valori ordinali dei limiti superiore e inferiori del tipo base devono
essere fra 0 e 255.

Un costruttore di insieme, che denota un valore di tipo set, e formato
scrivendo espressioni fra parentesi quadre. Ogni espressione definisce
il un valore dell'inisieme.

La notazione [ ] rappresenta l'insieme vuoto, che e' compatibile con tutti
i tipi di insieme.

 Esempi:

 { Tipi insieme }
   type
     Day = (Sun, Mon, Tue, Wed, Thu, Fri, Sat);
     CharSet = set of Char;
     Digits = set of 0..9;
     Days = set of Day;

 { Costruttori di insieme }
   ['0'..'9', 'A'..'Z', 'a'..'z', '_']
   [1, 5, I + 1 .. J - 1]
   [Mon..Fri]
