 Writeln (procedure)
 --------------------
Esegue la procedura Write, poi scrive in output un carattere di end-of-line
sul file.

 Dichiarazione:
 procedure Writeln([ var F: Text; ] P1 [, P2, ...,Pn ] );

 Note:
Writeln e' un'esensione della procedura write, per come e' definita sui
file di testo. Dopo aver eseguito la Write, Writeln scrive un carattere
di end-of-line (carriage-return/linefeed) sul file. Writeln(F) senza
altri paramentri aggiunge un ritorno a capo al file. (Writeln senza
la lista dei paramentri corrisponde a Writeln(Output).)

 Restrizioni:
Il file deve essere aperto per l'ouput.

 Vedere anche:
 Write (file di testo)
 Write (file tipati)

 Codice esempio:



 {Readln.PAS}

 {esempio per le procedure Readln e Writeln.}

 { Sotto Windows usare: }
 { uses WinCrt; }

 var
   s : String;
 begin
   Write('Enter a line of text: ');
   Readln(s);
   Writeln('You typed: ',s);
   Writeln('Hit  to exit');
   Readln;
 end.
