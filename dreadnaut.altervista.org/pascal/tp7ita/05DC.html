 absolute (direttiva standard)
 -------------------------------
Utilizzare la parola chiave absolute per dichiarare una variabile assoluta,
che cioe' risiede ad uno indirizzo di memoria specifico (assoluto).

 Sintassi:
 var Ident: type absolute Seg:Ofs;
 oppure
 var Ident: type absolute Variable;

 Note:
La prima forma dichiara espressamente l'indirizzo della variabile, usando
segmento e offset. Entrambe le costanti devono essere comprese fra
$0000 e $FFFF ( da 0 a 65535 ).

La seconda forma dichiara una nuova variabile che va a sovrapporsi ad
una gia' esistente, posizionandosi allo stesso indirizzo.

La lista di variabili nella dichiarazione puo' specificare solo un identificatore
quando e' presente la direttiva absolute.

Attenzione:
La prima forma dovrebbe essere usata con cautela. Quando Windows gira in
modalita' protetta, l'applicazione potrebbe non avere i diritti necessari ad
accedere a locazioni di memoria fuori dal programma. Tentare di accedere a
questa memoria probabilmente mandera' in crash il programma.

La seconda forma invece non da' problemi sotto Windows. La memoria a
cui si accede fa' sempre parte del programma.

Codice esempio:



 type
   VectorTable = array[0..255];

 var
  IntVectors: VectorTable absolute 0:0;
  CrtMode: Byte absolute $0040:$0049;
  Str: string;
  StrLen: Byte absolute Str;
