 FileSplit (function)     (WinDos unit e Dos unit)
 ---------------------------------------
Divide un nome di file nelle sue tre componenti.

 Dichiarazione:
 function FileSplit(Path, Dir, Name, Ext: PChar): Word

 Target:
Windows, Real, Protected.

 Note:
Divide il nome di file specificato in Path nelle sue tre componenti:
   -  Dir contiene il drive e il percorso di directory con ogni
      slash (\) iniziale e finale
   -  Name e' settata con il nome del file
   -  Ext contiene l'estensione preceduta da un punto

Se una delle stringhe per le componenti e' nil, la parte corrispondente del
path non viene scritta. Se il path non contiene una certa parte, la stringa
ritornata sara' vuota. Le lunghezze massime per le stringhe restituite in
Dir, Name ed Ext sono definite dalle costanti fsDirectory, fsFilename ed
fsExtension.

Il valore restituito e' una combinazione della bit-mask fcDirectory,
fcFilename ed fcExtension, ad indicare quali componenti erano presenti nel
path. Se il nome o l'estensione contengono un qualunque carattere wildcard
(simboli * o ?), il flag fcWildcards sara' settato nel valore restituito.

 Vedere anche:
 FileExpand
 FindFirst
 FindNext

 Codice esempio:



 {FileSplt.PAS}

 {Esempio per la funzione FileSplit.}

 { Sotto Windows usare: }
 { uses Strings, WinCrt, WinDos; }

 uses Strings, Dos;

 var
  Path: array[0..fsPathName] of Char;
  Dir: array[0..fsDirectory] of Char;
  Name: array[0..fsFileName] of Char;
  Ext: array[0..fsExtension] of Char;
 begin
  Write('Filename (WORK.PAS): ');
  Readln(Path);
  FileSplit(Path, Dir, Name, Ext);
  if Name[0] = #0 then StrCopy(Name, 'WORK');
  if Ext[0] = #0 then StrCopy(Ext, '.PAS');
  StrECopy(StrECopy(StrECopy(Path, Dir), Name), Ext);
  Writeln('Resulting name is ', Path);
 end.
