 Str (procedure)
 ----------------
Converte un valore numerico in una stringa.

 Dichiarazione:
 procedure Str(X [: Width [: Decimals ]]; var S:string);

 Target:
Windows, Real, Protected

 Note:
Converte il valore numerico X nella stessa rappresentazione a stringa che
Write produrrebbe in output.

 Vedere anche:
 Val
 Write (file di testo)
 Write (file tipati)

 Codice esempio:



 {Str.PAS}

 {Esempio per la procedura Str.}

 { Sotto Windows usare: }
 { uses WinCrt; }

 function IntToStr(I: Longint): String;
 { Converte qualunque integer in stringa }
 var
  S: string[11];
 begin
  Str(I, S);
  IntToStr := S;
 end;
 begin
  Writeln(IntToStr(-5322));
  Readln;
 end.
