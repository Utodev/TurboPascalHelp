 Val (procedure)
 ----------------
Converte un valore stringa nella sua rappresentazione numerica.

 Dichiarazione:
 procedure Val(S; var V; var Code: Integer);
dove:
  S     variabile di tipo stringa; deve essere una sequenza di caratteri
        che forma un unico numero (eventualmente con segno)
  V     variabile di tipo Integer o Real
  Code  variabile di tipo Integer.

 Target:
Windows, Real, Protected

 Note:
Converte il valora della stringa S nella sua rappresentazione numerica
come se venisse letta da un file di testo con Read.

 Vedere anche:
 Str

 Codice esempio:



 {Val.PAS}

 {Esempio per la procedura Val.}

 { For Windows: }
 { uses WinCrt; }

 var I, Code: Integer;
 begin
 { Prende il testo dalla linea di comando }
  Val(ParamStr(1), I, Code);
  { Errore durante la conversione a intero? }
  if code <> 0 then
    Writeln('Error at position: ', Code)
  else
    Writeln('Value = ', I);
  Readln;
 end.
