 FileExpand (function)     (WinDos unit e Dos unit)
 ----------------------------------------
Espande un nome di file in un nome completo di path.

 Dichiarazione:
 function FileExpand(Dest, Name: PChar): PChar;

 Target:
Windows, Real, Protected

 Note:
Espande il nome del file in Name facendolo diventare un nome completo.
Il nome ottenuno e' convertito in uppercase e consiste di:
   -  la lettera del disco
   -  un carattere due-punti (:)
   -  il percorso di directory relativo alla radice
   -  il nome del file 

Eventuali '.' e '..' nei riferimenti di directory sono rimossi e nome
ed estensione sono troncati rispettivamente ad 8 e 3 caratteri.
Il valore di ritorno e' scritto in Dest. Dest e Name possono riferirsi
alla stessa locazione.

Assumendo che il drive e la directory corrente siano C:\SOURCE\PAS, le
seguenti chiamate a FileExpand produrrebbero questi valori:

 FileExpand(S, 'test.pas') = 'C:\SOURCE\PAS\TEST.PAS'
 FileExpand(S, '..\*.TPW') = 'C:\SOURCE\*.TPW'
 FileExpand(S, 'c:\bin\turbo.exe') = 'C:\BIN\TURBO.EXE'

La funzione FileSplit puo' essere usata per dividere il risultato di
FileExpand in una stringa contenente drive e directory, una stringa
con il nome del file e una per l'estensione.

 File-attribute constants

 Codice esempio:



 {FileExp.PAS}

 {Esempio per la funzione FileExpand.}

 { Sotto Windows usare: }
 { uses WinDos, WinCrt; }

 uses Dos;

 const
  MyFile: PChar = 'TEST.FIL';

 var
  Where: PChar;

 begin
  GetMem(Where, 80);
  FileExpand(Where, MyFile);
  Writeln(MyFile, ' is in the current directory at ', Where, '.');
 end.
