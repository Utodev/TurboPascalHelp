 +--------------------+
 | Undefined external |
 +--------------------+

Una procedura o funzione dichiarata com EXTERN non ha una corrispondente
definizione PUBLIC all'interno di un file oggetto.

Verificare di aver inserito tutti i file oggetto attraverso direttive
{$L filename} e controllare l'ortografia
dei nomi delle funzioni e procedure nel file .ASM.
