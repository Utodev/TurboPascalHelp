 FindNext (procedure)     (Dos unit e WinDos unit)
 ----------------------------------------------------
Cerca la successiva occorrenza che corrisponde al nome e agli attributi
specificati in una precedente chiamata di FindFirst.

 Dichiarazione:
 procedure FindNext(var F: TSearchRec);

 Target:
Windows, Real, Protected

 Note:
Gli errori sono riportati in DosError. Il solo errore possibile e' 18 (non
ci sono altri file).

 Vedere anche:
 File-attribute constants
 FileExpand
 FindFirst
 TSearchRec

 Codice esempio:



 {Findfrst.PAS}

 {Esempio per le procedure FindFirst e FindNext.}

 { Sotto Windows usare: }
 { uses WinDos,WinCrt; }

 uses Dos;

 var
  DirInfo: SearchRec;       { Sotto Windows usare TSearchRec }
 begin                       { Sotto Windows usare faArchive }
  FindFirst('*.PAS', Archive, DirInfo); { uguale a DIR *.PAS }
  while DosError = 0 do
  begin
    Writeln(DirInfo.Name);
    FindNext(DirInfo);
  end;
 end.
