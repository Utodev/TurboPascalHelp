06AC
 
 Index of Dos Procedures and Functions (alphabetical)
 -----------------------------------------------------
Questo indice e' un elenco alfabeticamente ordinato delle funzioni e
procedure che e' possibile trovare nella unit Dos.

Per cercare routine Dos per categoria di funzionalita', vedere
l'indice categorico delle funzioni e procedure Dos.

 DiskFree    | Func | Restituisce il numero di byte liberi sul disco
             |      | specificato
 DiskSize    | Func | Restituisce la dimensione totale, in bytes, del
             |      | disco specificato
 DosExitCode | Func | Ritorna il valore di uscita di un sottoprocesso
 DosVersion  | Func | Restituisce il numero di versione DOS
 EnvCount    | Func | Restituisce il numero di variabili d'ambiente
             |      | DOS (environment strings)
 EnvStr      | Func | Restituisce la variabile d'ambiente specificata
 Exec        | Proc | Esegue il programma specificato con un dato
             |      | comando
 FExpand     | Func | Espande un nome di file in un nome completo di
             |      | path
 FindFirst   | Proc | Scandisce la directory specificata per un file
             |      | corrispondente
 FindNext    | Proc | Cerca un altro file dopo FindFirst
 FSearch     | Func | Cerca un file
 FSplit      | Func | Divide un nome di file nelle sue tre componenti
 GetCBreak   | Proc | Restituisce lo stato del controllo sul
             |      | Ctrl-Break sotto DOS
 GetDate     | Proc | Ritorna la data corrente del sistema operativo
 GetEnv      | Func | Restituisce il valore della variabile
             |      | d'ambiente specificata
 GetFAttr    | Proc | Restituisce gli attributi di un file
 GetFTime    | Proc | Restituisce la data e l'ora dell'ultima
             |      | operazione di scrittura sul file
 GetIntVec   | Proc | Restituisce l'indirizzo memorizzato nel vettore
             |      | di interrupt specificato
 GetTime     | Proc | Restituisce l'ora corrente del sistema
             |      | operativo
 GetVerify   | Proc | Restituisce lo stato del flag Verify sotto DOS
 Intr        | Proc | Esegue un interrupt software
 Keep        | Proc | Keep (o Terminate Stay Resident) termina il
             |      | programma mantenendolo in memoria
 MsDos       | Proc | Esegue una chiamata ad una funzione DOS
 PackTime    | Proc | Converte un record TDateTime
 SetCBreak   | Proc | Setta lo stato del controllo sul Ctrl-Break
             |      | sotto DOS
 SetDate     | Proc | Setta la data corrente del sistema operativo
 SetFAttr    | Proc | Modifica gli attributi di un file
 SetFTime    | Proc | Setta data e ora dell'ultima scrittura di un
             |      | file
 SetIntVec   | Proc | Setta un specifico vettore di interrupt
             |      | all'indirizzo dato
 SetTime     | Proc | Setta l'ora corrente del sistema operativo
 SetVerify   | Proc | Modifica lo stato della flag Verify sotto DOS
 SwapVectors | Proc | Scambia i puntatori SaveIntXX della unit System
             |      | con i vettori correnti
 UnpackTime  | Proc | Converte un Longint in un record TDateTime
