 MsDos (procedura)     (Dos unit e WinDos unit)
 -------------------------------------------------
Esegue una chiamata ad una funzione DOS

 Dichiarazione:
 procedure MsDos(var Regs: TRegisters);

 Target:
Windows, Real, Protected

 Note:
L'effetto di una chiamata a MsDos e' identico a quello di una chiamata a
Intr con IntNo uguale a $21. Registers e' un record definito nella unit
Dos. Se state scrivendo un programma windows usate TRegister.

 Restrinzioni:
Non usare interrupt software che:
 -  dipendono da uno specifico valore di SP o SS in entrata.
 -  modificano SP o SS in uscita.

 Vendere anche:
 Intr

 Codice esempio:



 {MsDos.PAS}

 { Esempio per la procedura MsDos }

 { Sotto Windows utilizzare: }
 { uses WinDos, WinCrt; }

 uses Dos;

 var
  date, year, month, day: string;
  regs: Registers;      { sotto Windows: TRegisters }

 begin
  regs.ah := $2a;
  with regs do
  msdos(regs);
  with regs do
    begin
      str(cx ,year);
      str(dh ,month);
      str(dl,day);
    end;
  date := month+'/'+day+'/'+year;
  writeln('s date is ', date);
 end.
