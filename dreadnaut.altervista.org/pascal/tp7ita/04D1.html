 Seek (procedure)
 -----------------
Sposta la posizione corrente in un file alla componente specificata.

 Dichiarazione:
 procedure Seek(var F; N: Longint);

 Target:
Windows, Real, Protected

 Note:
F e' una qualsiasi variabile di tipo file (escluso text), ed N e' una
espressione di tipo Longint. La posizione corrente nel file F e'
spostata alla componente numero N.  Il numero della prima componente
di un file e' 0.  Per espandere un file, potete spostarvi con Seek
una componente oltre la fine del file;  la chiamata Seek(F, FileSize(F))
sposta la posizione corrente nel file alla sua fine.

Con {$I-}, IOResult restituisce 0 se l'operazione ha avuto successo;
altrimenti ritorna un codice di errore diverso da zero.

 Restrinzioni:
 - Il file deve essere aperto.
 - Non puo' essere usata su file text.

 Vedere anche:
 FilePos

 Codice esempio:



 {FilePos.PAS}

 { Esempio per le funzioni FilePos e FileSize, e per
   la procedura Seek }

 { sotto Windows utilizzare: }
 { uses WinCrt;              }

 var
   f: file of Byte;
   size : Longint;
 begin
   { Il nome del file viene preso da linea di comando }
   Assign(f, ParamStr(1));
   Reset(f);
   size := FileSize(f);
   Writeln('File size in bytes: ',size);    { dimensione in byte }
   Writeln('Seeking halfway into file...'); { seek a meta' file  }
   Seek(f,size div 2);
   Writeln('Position is now ',FilePos(f));  { posizione corrente }
   Close(f);
 end.
