 +------------------------+
 | Unit file format error |
 +------------------------+

Per qualche motivo, il file .TPU non e' valido; verificare che sia
effettivamente un file .TPU.

La unit .TPU potrebbe essere stata creata con una versione obsoleta
del Turbo Pascal. In questo caso, un nuovo .TPU deve essere creato
compilando il corrispondente file .PAS.
