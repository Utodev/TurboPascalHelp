 Variabili di tipo SearchRec
 ----------------------------
Le procedure FindFirst e FindNext usano variabili di tipo SearchRec per
scandire le directory:

 type
   { record Search usato da FindFirst e FindNext }
   SearchRec = record
     Fill : array[1..21] of Byte;
     Attr : Byte;
     Time : Longint;
     Size : Longint;
     Name : string[12];
   end;

Le informazioni su ogni file trovato da FindFirst o FindNext sono
restituite in un SearchRec.

 Campo | Contenuto
 =========================================================================
 Attr  | Attributi del file (costruiti con le costanti di attributi file)
 Time  | Data e ora del file compattate (usare UnpackTime per scompattare)
 Size  | Dimensione del file, in byte
 Name  | Nome del file

Il campo Fill e' riservato dal DOS e non dovrebbe mai essere modificato.
