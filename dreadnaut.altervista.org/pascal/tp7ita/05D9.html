 Relational Operators / Operatori Relazionali
 --------------------------------------------

   Operator| Operation         |Result type|Operand types
 ==================================================================================
   =       | Uguale            | Boolean   | tipi semplici e compatibili, pointer,
           |                   |           | set, string o packed string
   <>      | Non uguale        | Boolean   | tipi semplici e compatibili, pointer,
           |                   |           | set, string o packed string
   <       | Minore di         | Boolean   | tipi semplici e compatibili, string,
           |                   |           | packed string o PChar
   >       | Maggiore di       | Boolean   | tipi semplici e compatibili, string,
           |                   |           | packed string o PChar
   <=      | Minore o uguale   | Boolean   | tipi semplici e compatibili, string,
           |                   |           | packed string o PChar
   >=      | Maggiore o uguale | Boolean   | tipi semplici e compatibili, string,
           |                   |           | packed string o PChar
   <=      | Sottoinsieme di   | Boolean   | tipi insieme compatibili
   >=      | Sovrainsieme di   | Boolean   | tipi insieme compatibili
   in      | Membro di         | Boolean   | Operando di sinistra: qualunque ordinale
           |                   |           | di tipo T.  Operando di destra: insieme 
           |                   |           | il cui tipo base e' compatibile con T.
