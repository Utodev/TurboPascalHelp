 TSearchRec (tipo)     (Dos unit e WinDos unit)
 ------------------------------------
Le procedure FindFirst e FindNext usano variabili di tipo TSearchRec per
scandire le directory.

 type
   TSearchRec = record
     Fill: array[1..21] of Byte;
     Attr: Byte;
     Time: Longint;
     Size: Longint;
     Name: array[0..12] of Char;
   end;

Le informazioni per ogni file trovato da una di queste procedure sono
restituite in un TSearchRec.

 - Attr contiene gli attributi del file (costruiti con le costanti di
   attributi file)
 - Time contiene la data e l'ora del file compattate (packed)
   (si puo' usare UnpackTime to scompattarle)
 - Size contiene la dimensione in bytes
 - Name contiene il nome
 - Fill e' riservata dal DOS e non dovrebbe mai essere modificata
