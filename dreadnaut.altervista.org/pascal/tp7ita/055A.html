 FilePos (function)
 -------------------
Ritorna la posizione corrente in un file.

 Dichiarazione:
 function FilePos(var F): Longint;

 Target:
Windows, Real, Protected

 Note:
F e' una variabile di tipo file. Se la posizione corrente e' all'inizio
del file, FilePos(F) restituisce 0.  Se la posizione corrente e' alla
fine del file (e cioe' se Eof(F) e' True) allora FilePos(F) e' uguale
a FileSize(F).

Con {$I-}, IOResult restituisce 0 se l'operazione ha avuto successo;
altrimenti ritorna un codice di errore diverso da zero.

 Restrinzioni:
 - Il file deve essere aperto.
 - Non puo' essere usata su file text.

 Vedere anche:
 FileSize
 Seek

 Codice esempio:



 {FilePos.PAS}

 { Esempio per le funzioni FilePos e FileSize, e per
   la procedura Seek }

 { sotto Windows utilizzare: }
 { uses WinCrt;              }

 var
   f: file of Byte;
   size : Longint;
 begin
   { Il nome del file viene preso da linea di comando }
   Assign(f, ParamStr(1));
   Reset(f);
   size := FileSize(f);
   Writeln('File size in bytes: ',size);    { dimensione in byte }
   Writeln('Seeking halfway into file...'); { seek a meta' file  }
   Seek(f,size div 2);
   Writeln('Position is now ',FilePos(f));  { posizione corrente }
   Close(f);
 end.
