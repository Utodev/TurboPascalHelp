 DosError (variabile)     (WinDos unit e Dos unit)
 ---------------------------------------------------
DosError e' usata dalla maggior parte delle procedure nelle unit Dos e
WinDos per riportare gli errori.

 var DosError: Integer;  {Variabile stato di errore}

I valori scritti in DosError sono codici di errore DOS.

Un valore 0 indica nessun errore; altri possibili codici di errore sono
i seguenti:

 Codice | Significato
 =====================================================
    2   | File not found       File non trovato
    3   | Path not found       Percorso non trovato
    5   | Access denied        Accesso negato
    6   | Invalid handle       Handle non valido
    8   | Not enough memory    Memoria insufficiente
   10   | Invalid environment  Ambiente non valido
   11   | Invalid format       Formato non valido
   18   | No more files        Non ci sono altri file
