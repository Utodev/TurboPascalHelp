 NormVideo (procedure)     (Crt unit)
 -------------------------------------
Ripristina gli attributi iniziali del testo, presi dalla posizione
del cursore all'avvio del programma.

 Dichiarazione:
 procedure NormVideo;

 Target:
Real, Protected

 Note:
C'e' una variabile Byte nella unit Crt --TextAttr-- che mantiene gli
attributi video correnti. Normvideo ripristina TextAttr al valore che
aveva quando il programma e' stato lanciato.

 Vedere anche:
 HighVideo
 LowVideo
 TextBackground
 TextColor

 Codice esempio:



  {NormVid.PAS}

 { Esempio per NormVideo, TextBackGround e TextColor }

 uses Crt;
 begin
   { Caratteri verdi su nero }
   TextColor(Green);
   TextBackground(Black);
   WriteLn('Hey there!');
   { Caratteri rosso chiaro lampeggianti su grigio }
   TextColor(LightRed+Blink);
   TextBackground(LightGray);
   WriteLn('Hi there!');
   { Caratteri gialli su fondo blu }
   TextColor(14); { Giallo = 14 }
   TextBackground(Blue);
   WriteLn('Ho there!');
   NormVideo; { Attributi originali }
   WriteLn('Back to normal...');
 end.
