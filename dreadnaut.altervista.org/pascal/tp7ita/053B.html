 Copy (function)
 ----------------
Restituisce una sottostringa di una stringa

 Dichiarazione:
 function Copy(S: String; Index: Integer; Count: Integer): String;

 Target:
Windows, Real, Protected

 Note:
S e' un'espressione di tipo stringa. Index e Count sono espressione di tipo
intero. Copy restituisce una stringa contenente Count caratteri partendo
dall'Index-esimo carattere di S.

Se Index e' piu' grande della lunghezza di S, Copy ritorna una stringa vuota.
Se Count specifica piu' caratteri di quanti ne rimangano dopo l'Index-esima
posizione, sono restituiti soltanto i caratteri rimanenti.

 Vedere anche:
 Concat
 Delete
 Insert
 Length
 Pos

 Codice esempio:



 {Copy.PAS}

 {Esempio per la funzione Copy.}

 var S: String;
 begin
   S := 'ABCDEF';
   S := Copy(S, 2, 3); { 'BCD' }
 end.
