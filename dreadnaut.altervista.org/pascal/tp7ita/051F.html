 Write (procedure)
 ------------------
Per file tipati, scrive una variabile in una componente del file.
Per file di testo, scrive uno o piu' valori nel file.

 Dichiarazione:
 File tipati:   procedure Write(F, V1 [, V2,...,Vn ] );
 File di testo: procedure Write( [ var F: Text; ] P1 [,P2,...,Pn ] );

 Target:
Windows, Real, Protected

 Note:
Il file deve essere aperto per l'output.

F se specificata e' una variabile di tipo file di testo.

 File di testo
Ogni P e' un parametro di scrittura che include un espressione di output
il cui valore sara' scritto nel file.

Un parametro puo' anche contentere le specifiche di larghezza del campo
e il numero di cifre decimali.

Ogni espressione di output deve essere di tipo Char, Integer, Real, string,
packed string o Boolean.

 File tipati
Ogni V e' una variabile dello stesso tipo delle componenti di F.

Per ogni variabile scritta, la posizione corrente nel file e' spostata in
avanti alla componente successiva.

Se la posizione corrente e' la fine del file, il file viene espanso.

 Valori di ritorno:
{$I-} IOResult restituisce 0 se l'operazione ha avuto successo,
      altrimenti un codice di errore diverso da zero.

 Vedere anche:
 Read
 Readln
 Writeln

 Codice esempio:



  {Eof.PAS}

 {Esempio per le funzione Eof, Read, and Write function (file di testo).}

 { Sotto Windows usare: }
 { uses WinCrt, WinDos; }

 var
   F: Text;
   Ch: Char;
 begin
   { Prende il file da leggere dalla line di comando }
   Assign(F, ParamStr(1));
   Reset(F);
   while not Eof(F) do
   begin
     Read(F, Ch);
     Write(Ch);  { Visualizza il file di testo }
   end;
 end.
