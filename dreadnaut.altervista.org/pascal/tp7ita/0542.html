 Delete (procedura)
 -------------------
Cancella una sottostringa da una stringa.

 Dichiarazione:
 procedure Delete(var S: String; Index: Integer; Count:Integer);

 Target:
Windows, Real, Protected

 Note:
S e' una variabile di tipo stringa. Index e Count sono espressioni di tipo
intero. Delete elimina Count caratteri da S a partire dalla Index-esima
posizione. Se Index e' piu' grande della lunghezza di S, non viene cancellato
nessun carattere. Se Count specifica piu' caratteri di quanti ne rimangano
nella stringa a partire dall'Index-esima posizione, la parte restante della
stringa da questo punto in avanti viene cancellata.

 Vedere anche:
 Concat
 Copy
 Insert
 Length
 Pos

 Codice esempio:



 {Delete.PAS}


 {Esempio per la procedura Delete.}

 { Sotto Windows: }
 { uses WinCrt;   }

  var
    s: string;
  begin
    s := 'Honest Abe Lincoln';
    Delete(s,8,4);
    Writeln(s); { 'Honest Lincoln' }
  end.
